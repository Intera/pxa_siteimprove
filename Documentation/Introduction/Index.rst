.. _what-it-does:

What does it do?
----------------

An integration of Siteimprove (https://siteimprove.com/) into TYPO3 CMS. A modal will be available in the
Page Module for site and page analysis.

.. _deeplinking-support:

Deeplinking Support
-------------------

The *Siteimprove Intelligence Platform* supports CMS Deeplinking. Clicking on an *Edit in CMS* button within the Siteimprove platform will take you to the corresponding page within TYPO3's Page Module.
